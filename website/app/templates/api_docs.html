API Reference

Overview

The REST API lets you programmatically browse and download ArduPilot Lua scripts. All responses are JSON (UTF-8) and the API is read-only with no authentication required.

Base URL /api/v1
Format JSON (UTF-8)
Authentication None — read-only public API

An interactive explorer (Swagger UI) is also available at /api/docs.

GET/api/v1/health

Returns the health status of the API and the total number of indexed scripts.

Response
{
  "status": "ok",
  "script_count": 42
}

GET/api/v1/types

Returns the count of indexed scripts broken down by category.

Response
[
  { "type": "applets", "count": 10 },
  { "type": "drivers", "count": 5 },
  { "type": "tools",   "count": 3 }
]

GET/api/v1/scripts

Returns a paginated list of scripts with optional search and filter parameters.

Query parameters
Parameter Type Default Description
q string — Full-text search on name and short_description
type string — Filter by category: applets, drivers, or tools
vehicle string — Filter by target vehicle: copter, plane, rover, etc.
page integer 1 Page number (1-based)
limit integer 50 Results per page (max 200)
Response
{
  "total": 18,
  "page": 1,
  "limit": 50,
  "results": [
    {
      "name": "HelloWorld",
      "type": "applets",
      "short_description": "Minimal example applet",
      "version": "1.0.0",
      "date": "2026-05-18",
      "vehicle": null,
      "min_firmware": null,
      "description": null
    }
  ]
}

GET/api/v1/scripts/{type}/{name}

Returns full metadata and the documentation body for a single script.

Path parameters
Parameter Description
type Script category: applets, drivers, or tools
name Script stem (filename without .lua), e.g. HelloWorld
Response
{
  "name": "HelloWorld",
  "type": "applets",
  "short_description": "Minimal example applet",
  "version": "1.0.0",
  "date": "2026-05-18",
  "vehicle": null,
  "min_firmware": null,
  "description": "## Description\n\nSends a GCS message every 5 seconds."
}
Error responses
Status Meaning
404 Script not found

GET/api/v1/scripts/{type}/{name}/raw

Downloads the raw Lua source file. Response Content-Type is text/plain and Content-Disposition suggests saving as {name}.lua.

Path parameters
Parameter Description
type Script category
name Script stem
Example
curl https://your-instance/api/v1/scripts/applets/HelloWorld/raw \
  -o HelloWorld.lua
Error responses
Status Meaning
404 Script or Lua file not found
